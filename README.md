# プライバシーポリシー

本プライバシーポリシーは、Takeshi Iwata（以下、「開発者」とします）が提供するスマートフォン向けアプリケーション（以下、「本アプリ」とします）における、ユーザーの個人情報の取り扱いについて定めるものです。

## 1. 個人情報の収集と利用について

本アプリでは、ユーザーの氏名、メールアドレス、電話番号などの個人を特定できる情報を直接収集することはありません。

## 2. データの保存について

本アプリ内でユーザーが入力したデータや選択肢、統計情報などは、すべてユーザーの端末内（ローカルデータベース）にのみ保存されます。これらのデータが外部のサーバーに送信されることや、開発者が取得・閲覧することは一切ありません。

## 3. サードパーティサービスについて

本アプリは、アプリ内課金（投げ銭等）の処理を目的として、Apple Inc. または Google LLC が提供する決済システムを利用しています。これに伴う決済情報の処理および管理は、各プラットフォームの利用規約およびプライバシーポリシーに従って行われます。開発者がユーザーのクレジットカード情報などの決済情報を取得することはありません。

## 4. 広告について

本アプリには、第三者による広告表示（バナー広告、動画広告など）は一切含まれておりません。そのため、広告配信を目的とした行動履歴等のデータ収集も行われません。

## 5. 免責事項

本アプリの利用により生じたトラブルや損害について、開発者は一切の責任を負いかねます。あらかじめご了承ください。

## 6. プライバシーポリシーの変更

開発者は、法令の変更やアプリのアップデートに伴い、本プライバシーポリシーを事前の通知なく変更することがあります。変更後のプライバシーポリシーは、本ページに掲載された時点から効力を生じるものとします。
